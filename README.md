# Portfólio Luiz DEV — Estilo e Diretrizes

## Paleta de cores
- Fundo (`--color-bg`): `#ffffff`
- Superfície (`--color-surface`): `#ffffff`
- Elevação leve (`--color-elev`): `#f5f7fa`
- Primária (`--color-primary`): `#2563eb`
- Acento (`--color-accent`): `#0ea5e9`
- Borda (`--color-border`): `#e5e7eb`
- Texto (`--color-text`): `#111827`
- Texto secundário (`--color-muted`): `#6b7280`
- Foco/realce (`--color-focus`): `#93c5fd`

## Fontes escolhidas
- Base: `Inter` (Google Fonts)
- Fallbacks: `system-ui, -apple-system, Segoe UI, Roboto, Ubuntu, "Helvetica Neue", Arial, sans-serif`
- Ícones: Font Awesome (`@6.4.0`)

## Estilo adotado (breve)
- Tema claro e minimalista com amplo uso de espaços em branco.
- Tipografia legível (Inter), hierarquia simples e contrastes equilibrados.
- Componentes com bordas finas, sombras sutis e animações discretas (ex.: sublinhado na navegação).
- Layout responsivo (mobile-first) e atenção à acessibilidade (ARIA, `aria-current`, foco visível).
